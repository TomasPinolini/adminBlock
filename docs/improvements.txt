===============================================================================
 ALTA PRIORIDAD
===============================================================================

    1. Sistema de Roles y Permisos
       - ROL ADMIN: Papa, Tio - acceso completo
       - ROL OPERADOR: Mama, Hermana - solo crear clientes/pedidos
       - Restringir acceso a configuracion, cotizacion y reportes
       - Middleware de autorizacion por ruta

    2. Auditoria / Log de Actividad  [COMPLETADO]
       - Registrar quien hizo cada accion (crear, editar, eliminar)
       - Registrar que cambio (campo anterior -> nuevo valor)
       - Tabla activity_logs con userId, action, entity, entityId, changes, timestamp
       - UI para ver historial de actividad por entidad (ActivityModal)
       - Integrado en pedidos: crear, editar, eliminar, duplicar, archivar, pagos

===============================================================================
 MEDIA PRIORIDAD
===============================================================================

    3. Generacion de PDFs
       - Generar PDF de facturas (Factura A, C) desde pedidos
       - Generar PDF del reporte mensual (ventas + gastos + balance)
       - Libreria sugerida: @react-pdf/renderer o jsPDF

    4. Calendario de Entregas
       - Vista calendario de pedidos por fecha de entrega (dueDate)
       - Vista semanal y mensual
       - Drag & drop para reprogramar entregas

    5. Dashboard Mejorado
       - Graficos de ventas vs gastos por mes (tendencia)
       - Top clientes por facturacion
       - Servicios mas vendidos
       - Comparacion mes actual vs mes anterior
       - Libreria sugerida: recharts

    6. Operaciones en Lote (Bulk Actions)
       - Seleccionar multiples pedidos -> archivar, cambiar estado
       - Seleccionar multiples clientes -> exportar, etiquetar
       - Checkbox en listas + barra de acciones flotante

    7. Busqueda Global
       - Barra de busqueda universal (Ctrl+K / Cmd+K)
       - Buscar en clientes, pedidos, cotizaciones, proveedores
       - Resultados agrupados por tipo con navegacion rapida
       - Libreria sugerida: cmdk
       NOTA: Buscador por cliente en cotizaciones ya implementado

    8. Gastos Recurrentes
       - Marcar gastos como recurrentes (mensual: alquiler, EPE, etc.)
       - Auto-generar gastos recurrentes al inicio de cada mes
       - Editar/cancelar recurrencia

    9. IVA Compras (Credito Fiscal)
       - Agregar campo taxAmount a order_materials / supplier_materials
       - Calcular IVA compras mensual automaticamente
       - Posicion IVA mensual: IVA Ventas - IVA Compras = deuda AFIP
       - Vista en reportes mensuales

 BAJA PRIORIDAD / FUTURO

        - Link publico para que el cliente vea estado de su pedido

    13. Plantillas de Pedidos
        - Guardar pedidos tipo como templates

    14. Historial de Comunicacion por Cliente
        - Log de mensajes enviados (WhatsApp, Instagram)
        - Timeline por cliente: pedidos + mensajes + pagos
        - Notas rapidas por interaccion

    15. Notificaciones In-App
        - Centro de notificaciones (campanita)
        - Pedidos vencidos, pagos pendientes, entregas hoy
        - Marcar como leida / ver todas

        - Ctrl+K: busqueda global
        - N: nuevo pedido / cliente segun contexto

    17. Exportacion Avanzada  [COMPLETADO]

        - Backup completo de datos en JSON/CSV

    18. Integracion con AFIP

        - Consulta de CUIT

        - Requiere certificado digital

===============================================================================
 OPTIMIZACIONES TECNICAS (cuando escale el negocio)
===============================================================================

    Estas mejoras son para cuando el volumen de datos crezca significativamente.

    - Paginacion cursor-based (cuando > 1000 registros por entidad)
    - N+1 query optimization con Drizzle 'with' eager loading
    - Rate limiting en APIs (cuando haya riesgo de abuso)
    - Structured logging con Pino/Winston
    - Code splitting / dynamic imports para modals pesados
    - HTTP caching strategy (Cache-Control headers)
    - CSRF protection (cuando app sea mas publica)
    - Optimistic updates en React Query
    - Retry logic para requests fallidos
    - Habilitar TypeScript strict mode
    - IVA rate configurable desde settings (actualmente hardcoded 21%)
    - Estandarizar manejo de fechas (timezone-aware)
